## Strategy

1. **Step 1**: Problem approach
   - 가중치가 없는 방향 그래프에서 모든 정점 쌍 `(i, j)`에 대해 도달 가능한지(경로가 존재하는지) 여부를 판별해야 함
   - '모든 정점에서 모든 정점'으로의 경로를 확인해야 하므로 **플로이드-워셜(Floyd-Warshall)** 알고리즘 
2. **Step 2**: Optimization points
   - 최단 거리를 구하는 것이 목적이 아니므로, 거리를 더하는 과정에서 오버플로우가 나지 않도록 연결되지 않은 간선은 적당히 큰 값(`INF = 100`)으로 초기화
   - 탐색 종료 후 `adj[i][j] < INF` 라면 도달 가능(1), 그렇지 않다면 도달 불가능(0)으로 치환하여 출력

<br>

## Solution

```java
import java.io.*;
import java.util.*;

public class BOJ11403 {

    public static void main(String[] args) throws IOException {

        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringBuilder sb = new StringBuilder();
        StringTokenizer st;

        int N = Integer.parseInt(br.readLine());
        int INF = 100;
        int[][] adj = new int[N][N];

        for (int i = 0; i < N; i++) {
            st = new StringTokenizer(br.readLine());
            for (int j = 0; j < N; j++) {
                adj[i][j] = Integer.parseInt(st.nextToken()) == 1 ? 1 : INF;
            }
        }

        // 플로이드-워셜
        for (int k = 0; k < N; k++) {
            for (int i = 0; i < N; i++) {
                for (int j = 0; j < N; j++) {
                    adj[i][j] = Math.min(adj[i][j], (adj[i][k] + adj[k][j]));
                }
            }
        }

        // 결과 변환 및 출력
        for (int i = 0; i < N; i++) {
            for (int j = 0; j < N; j++) {
                int data = adj[i][j] < INF ? 1 : 0;
                sb.append(data).append(" ");
            }
            sb.append("\n");
        }

        System.out.println(sb);
    }
}
```
